Sedikit Tentang Profile Saya

Halo! Saya Tiyas Frahesta, seorang mahasiswa di Politeknik Mardira dari program studi Rekayasa Perangkat Lunak. Saya membuat web portofolio ini sebagai wadah untuk memperkenalkan diri secara profesional sekaligus mendokumentasikan berbagai proyek, pengalaman, dan keterampilan yang saya miliki di dunia teknologi.

Saya memiliki ketertarikan besar dalam berbagai bidang teknologi, mulai dari networking, programming, hingga hardware. Kecintaan saya terhadap dunia IT mendorong saya untuk terus belajar, bereksperimen, dan mengembangkan kemampuan baik secara akademis maupun praktikal.Melalui web ini, saya berharap dapat berbagi perjalanan saya sebagai seorang mahasiswa dan praktisi IT, sekaligus menjadi jembatan untuk terhubung dengan rekan-rekan, mentor, maupun peluang di industri teknologi.
